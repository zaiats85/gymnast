=== MDLWP Portfolio ===
Requires: mdlwp.com
Author: Brad Williams - http://braginteractive.com

== Description ==

This plugin enables a portfolio layout for MDLWP.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upzip `mdlwp-portfolio.zip`
2. Move `archive-portfolio.php` and `single-portfolio.php` to the root directory of MDLWP(/wp-content/themes/mdlwp/)
3. Upload or FTP the `mdlwp-portfolio` folder into the plugin directory (/wp-content/plugins/)
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add Portfolio items in the Portfolio custom post type

For further details please watch the videos on this plugin at http://youtube.com/braginteractive
